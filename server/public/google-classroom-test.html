Google Classroom Add-on Test Page

Note: This is a test page to demonstrate the Google Classroom add-on functionality. In a real implementation, these iframes would be loaded within Google Classroom.

Add-on Discovery (Attachment Setup)

This iframe simulates the Google Classroom add-on discovery view where teachers select resources to attach to assignments.

Available Endpoints

Here are the Google Classroom add-on endpoints that have been implemented:

OAuth & Authentication:

  • /google-classroom/oauth/login - OAuth login redirect
  • /google-classroom/oauth/callback - OAuth callback handler

Add-on Iframes:

  • /google-classroom/addon-discovery - Resource selection iframe
  • /google-classroom/teacher-view - Teacher view iframe
  • /google-classroom/student-view - Student view iframe
  • /google-classroom/student-work-review - Student work review iframe

API Endpoints:

  • /google-classroom/api/v1/jwt/portal - Portal JWT generation
  • /google-classroom/api/v1/jwt/firebase - Firebase JWT generation
  • /google-classroom/api/v1/classes/{courseId} - Class information
  • /google-classroom/api/v1/offerings/{courseId} - Offering information

Resource Endpoints:

  • /google-classroom/resource-launch - Resource launch handler
  • /google-classroom/select-resource - Resource selection handler
  • /google-classroom/courses/{courseId}/roster - Course roster
  • /google-classroom/profile - User profile

Required Environment Variables

Make sure you have set the following environment variables:

  • GOOGLE_CLIENT_ID - Your Google OAuth client ID
  • GOOGLE_CLIENT_SECRET - Your Google OAuth client secret
  • PUBLIC_URL - Your public server URL (must be HTTPS)
